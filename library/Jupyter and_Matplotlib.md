## *📚 What is a Python Library?*

A Python library is a collection of pre-written code that helps solve common problems.

👉 Instead of writing everything from scratch, you can reuse functions.

```python
#Example:
import math
print(math.sqrt(16))
```
```python
✔ Output:

4.0
```
